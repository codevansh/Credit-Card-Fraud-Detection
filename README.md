# Credit Card Fraud Detection Model

This model takes an input CSV File to detect and check any Fraud transactions happened in the system. This is really important for the banking sector to check for any Fraud Entry even if the Chances are very less.

The dataset was downloaded from kaggle: 

This dataset was cut down to around 50,000 entries which was much easier to interpret for a model.
